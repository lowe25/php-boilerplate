# PHP MVC Boilerplate

A lightweight PHP MVC boilerplate using **Twig**, **TailwindCSS**, and **TypeScript**. Powered by XAMPP
Ideal for creating modern PHP projects with organized front-end and back-end code.

---

## Features

- **MVC architecture** for PHP projects
- **Twig templates** for flexible front-end rendering
- **TailwindCSS** for utility-first styling
- **TypeScript** with ESBuild bundler for modern JS
- **Live reload** for CSS/JS changes using BrowserSync

---

## Clone the repository:

```bash
git clone https://github.com/your-username/php-boilerplate.git
cd php-boilerplate
```

## Installation

- npm install

## Start the development server

- npm run dev

## Note

Do this if having a problem in Live reload

- Ctrl + Shift + R
